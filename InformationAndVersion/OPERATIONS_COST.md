## 2. 变动成本 (Variable Costs)

### 2.1 AI API 调用费用 (DeepSeek)
目前使用的是 DeepSeek API (deepseek-chat 模型)。
*   **输入价格**: ¥1.00 / 百万 tokens
*   **输出价格**: ¥2.00 / 百万 tokens

**估算模型**：
*   假设每个活跃用户每天请求 3 次 AI 建议。
*   每次请求消耗约 500 tokens (Prompt + 回复)。
*   1000 个活跃用户每月的成本：
    $$ 1000 \text{人} \times 3 \text{次} \times 30 \text{天} \times 500 \text{tokens} = 45,000,000 \text{tokens} $$
    $$ 45 \text{百万} \times ¥1.5 (\text{均价}) \approx ¥67.5 / \text{月} $$
*   **结论**：在用户量达到 10 万级别之前，AI 成本几乎可以忽略不计。

### 2.2 服务器/云存储
*   **当前状态**: ¥0
    *   V1.0 版本采用 **Local-First** 架构，不使用后端服务器。
    *   用户数据存储在本地或 iCloud (由 Apple 免费提供给用户)。
*   **未来扩展**:
    *   如果自建后端 (Node.js + Postgres)，需购买云主机 (ECS/轻量应用服务器)，起步价约 ¥60/月。

## 3. 隐性成本 (Hidden Costs)

*   **开发维护**: 你的时间成本 (Bug修复、新功能开发)。
*   **客服支持**: 处理 GitHub Issues 或邮件反馈的时间。
*   **推广营销**: 如果要在小红书/抖音投放广告，预算无上限。
