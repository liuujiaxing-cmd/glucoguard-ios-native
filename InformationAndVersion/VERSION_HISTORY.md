# 版本管理记录 (Version History)

## v1.0.0 (Initial Release)
**发布日期**: 2024-02-04
**Build**: 1

### 主要功能
*   [x] **基础记录**: 记录血糖数值、时间、类型（空腹/餐后）及备注。
*   [x] **智能输入**:
    *   集成 **Vision OCR**，支持拍照识别血糖仪读数。
    *   集成 **Speech** 框架，支持语音转文字记录。
*   [x] **数据展示**:
    *   首页今日概览（平均值、测量次数）。
    *   动态血糖趋势折线图。
    *   最近记录列表。
*   [x] **AI 服务**:
    *   接入 **DeepSeek API**，提供基于用户数据的个性化健康建议。
    *   本地规则引擎兜底，断网可用。
*   [x] **药物管理**:
    *   支持添加多种药物、剂量及服用时间。
    *   首页打卡日历。
    *   本地推送通知提醒。
*   [x] **用户系统**:
    *   Onboarding 引导页（收集年龄/病史）。
    *   游客模式与 UserProfile 本地持久化。
*   [x] **UI/UX**:
    *   全新 **GlucoGuard** 品牌设计。
    *   医疗蓝/莫兰迪色系主题 (`AppTheme`)。
    *   全屏沉浸式记录页，大字体适配老年人。

### 已知问题 / 待优化
*   iCloud 同步目前仅记录日志，尚未完全实装多设备漫游。
*   暂不支持导出 PDF 报告（目前仅支持 CSV）。
*   OCR 识别在复杂光线下可能偶尔识别失败。
